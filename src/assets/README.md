# Assets

Este diretório contém os recursos estáticos, como imagens, ícones e fontes locais, consumidos diretamente pelas telas do aplicativo CarbonEye.

A pasta principal de ícones do sistema Expo está localizada no diretório raiz `./assets` para compatibilidade de build nativo, enquanto imagens e logotipos personalizados adicionais podem ser armazenados aqui.
